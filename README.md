# Data Analyzer and Transformer

## Overview

The Data Analyzer and Transformer is a Python console-based application
designed to perform basic operations on numerical data stored in a 1D
list. It provides an interactive menu-driven interface where users can
input data and perform analysis such as summary statistics, sorting,
filtering, and factorial calculation.

## Features

-   Input and store numerical data in a list
-   Display complete data summary (count, minimum, maximum, sum,
    average)
-   Calculate factorial of a given number using recursion
-   Filter dataset based on a threshold value
-   Sort data in ascending or descending order
-   Display dataset statistics separately
-   Simple menu-driven user interaction

## Technologies Used

-   Python
-   Data Structures:
    -   List

## How It Works

1.  The program starts with an empty dataset
2.  User selects options from the menu:
    -   1 → Input Data
    -   2 → Display Data Summary
    -   3 → Calculate Factorial
    -   4 → Filter Data
    -   5 → Sort Data
    -   6 → Display Dataset Statistics
    -   7 → Exit program
3.  The program continues until the user selects Exit

## Notes

-   Input must be space-separated integers only
-   Operations like summary, sorting, and filtering require non-empty
    data
-   Factorial is implemented using recursion, so very large inputs may
    cause recursion limit errors
-   Filtering returns values greater than or equal to the given
    threshold
